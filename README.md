# PySide6 + Folium Map Applications

A collection of desktop applications that combine PySide6 and Folium to create interactive leaflet-based map interfaces.

## Applications Included

### 1. Basic Map Application (`map_app.py`)
- Simple, clean interface for basic map viewing
- Essential mapping features without complexity

### 2. Enhanced Map Application (`enhanced_map_app.py`)
- Advanced features with tabbed interface
- Multiple layers, shapes, heatmaps, and clustering
- Professional-grade mapping capabilities

### 3. Advanced Features Example (`advanced_features.py`)
- Generates standalone HTML with advanced folium features
- Demonstrates capabilities for integration

### 4. Application Launcher (`launcher.py`)
- Easy-to-use launcher for all applications
- Choose the right tool for your needs

## Key Features

### Basic Features
- **Interactive Map Display**: Uses folium to generate leaflet maps displayed in a PySide6 QWebEngineView
- **Multiple Map Styles**: Choose from various tile layers including OpenStreetMap, CartoDB, and Stamen maps
- **Location Navigation**: Enter latitude/longitude coordinates to navigate to specific locations
- **Zoom Control**: Adjust zoom level from 1-18
- **Marker Management**: Add markers, clear all markers, load sample data
- **Real-time Updates**: Map updates instantly when settings change

### Enhanced Features
- **Tabbed Interface**: Organized controls across Basic, Features, and Data tabs
- **Advanced Layers**: Minimap, fullscreen control, measure tools, drawing tools
- **Shape Drawing**: Add circles and polygons with custom colors and sizes
- **Marker Clustering**: Group nearby markers for better visualization
- **Heatmap Visualization**: Generate and display heat data with adjustable intensity
- **Sample Datasets**: Load US cities or world capitals with one click
- **Statistics Panel**: Real-time statistics about map elements
- **Color Picker**: Custom color selection for shapes and markers

## Requirements

- Python 3.7+
- PySide6
- folium

## Installation

1. Create a virtual environment:
```bash
python -m venv .venv
```

2. Activate the virtual environment:
```bash
# Windows
.venv\Scripts\activate

# macOS/Linux
source .venv/bin/activate
```

3. Install dependencies:
```bash
pip install -r requirements.txt
```

## Controls Guide

### Basic Application Controls
1. **Latitude/Longitude**: Enter coordinates to center the map
2. **Zoom**: Set the zoom level (1-18)
3. **Map Style**: Choose from different tile layer styles
4. **Update Map**: Refresh the map with current settings
5. **Add Marker**: Add a marker at the current center coordinates
6. **Clear Markers**: Remove all markers from the map
7. **Load Sample Data**: Load sample markers for major US cities

### Enhanced Application Controls

#### Basic Tab
- Same as basic application controls

#### Features Tab
- **Map Layers**: Toggle minimap, fullscreen, measure tools, drawing tools
- **Shapes**: Add circles and polygons with custom radius and colors
- **Clustering**: Enable marker clustering for better organization

#### Data Tab
- **Sample Data**: Load US cities or world capitals
- **Heatmap**: Generate random heat data and control intensity
- **Statistics**: View real-time statistics about map elements

### Default Location

The application starts centered on San Francisco, CA (37.7749, -122.4194) with zoom level 10.

## Technical Details

- Uses `QWebEngineView` to display the HTML map generated by folium
- Temporary HTML files are created and cleaned up automatically
- Markers are stored in memory and persist across map updates
- Supports all standard folium map styles and features
